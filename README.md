# 🏍️ Sistema de Gestão - Isaias Motos

Sistema web desenvolvido para gerenciamento operacional da oficina Isaias Motos.

---

# 📋 Sobre o Projeto

A plataforma foi criada para auxiliar no controle de:

- Clientes
- Motocicletas
- Ordens de Serviço
- Peças
- Estoque
- Serviços realizados

O sistema centraliza todas as informações da oficina em uma única plataforma.

---

# 🚀 Funcionalidades

## 👤 Clientes
- Cadastro de clientes
- Consulta de clientes
- Atualização de dados
- Exclusão de clientes

## 🏍️ Motos
- Cadastro de motocicletas
- Vínculo com proprietário
- Histórico de serviços

## 🔧 Ordens de Serviço
- Abertura de ordem
- Controle de status
- Registro de peças
- Registro de serviços

## 📦 Estoque
- Controle de peças
- Baixa automática no estoque
- Alerta de estoque mínimo

---

# 🛠️ Tecnologias Utilizadas

# 🗄️ Banco de Dados

- Chaves primárias
